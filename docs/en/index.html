# What is Reposilite?

  • Repository manager for Maven artifacts
  • Replaces managers like Nexus, Archiva or Artifactory
  • Open source project

# Why?

  • Reduce usage of your resources to even 8MB of RAM
  • 95%+ test coverage
  • Easy to use
